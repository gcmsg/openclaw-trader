## ✅ 可自动执行（做完通知主人即可）

### Bug 修复与引擎优化
- 修复代码报错、异常、边界条件 bug
- 修复测试失败
- 修复 shell 命令、cron 表达式、路径等配置错误
- 依赖包版本冲突、兼容性问题
- **在实际使用交易引擎（testnet/paper）过程中发现的问题，及时修复和优化，无需等待授权**
  - 包括：信号逻辑缺陷、执行路径异常、风控边界不准确、通知缺失、滑点/手续费计算偏差等
  - 修复后附上问题描述、修复方案、测试结果，通过 Telegram 汇报

### 交易策略优化
- 调整技术指标参数（MA 周期、RSI 阈值、MACD 参数等）
- 根据复盘数据微调止损/止盈百分比
- 调整情绪门控阈值（恐惧贪婪指数边界值）
- 优化信号过滤条件（减少假信号）
- 修改 `config/strategy.yaml` 中的参数

### 代码质量
- 重构、清理冗余代码
- 补充或修正单元测试
- 性能优化（减少 API 调用、降低延迟）
- 修复日志格式、补充错误处理

### 文档
- 更新 README（中英双语同步更新）
- 更新 skill/（SKILL.md + references/）与 README 同等级别实时同步
- 更新代码注释

## 🤖 模拟盘完全自主授权（2026-03-01 主人授权）

**现货 testnet、合约 testnet、paper 模拟盘的所有交易操作，无需任何事前授权，执行后即时报告。**

包括但不限于：
- 自主开多、开空、平仓、调仓、止损/止盈调整
- 修改策略配置和参数
- 启动/停止相关服务和场景

汇报规则：
- **即时报告**：每次开仓/平仓/止损触发后立刻发 Telegram 通知
- 按时发送复盘报告（每日 09:00 + 21:00）

风控底线（不可突破）：
- 单笔止损 ≤ 5%，总亏损 20% 自动暂停
- 不得影响实盘账户，不得跨越 testnet/paper 边界
